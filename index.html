Code-Schema
YouTube-Quelle für den Zufallsgenerator

Kostenlos: Google Cloud Console → Projekt anlegen → „YouTube Data API v3“ aktivieren → API-Key erstellen. Wird nur lokal in deinem Browser gespeichert, nie irgendwohin verschickt.

Steht in der Playlist-URL hinter list=. Voreingestellt: "1000 Greatest Songs Of All Time" (983 Songs) — kann durch jede beliebige öffentliche Playlist ersetzt werden.

Spotify-Anbindung (Jahr + exakter Track-Link)

Kostenlos: developer.spotify.com/dashboard → App erstellen ("Development Mode" reicht, dafür braucht dein Spotify-Account ein aktives Premium-Abo) → Client-ID & Client-Secret kopieren. Beides wird nur lokal in deinem Browser gespeichert. Achtung: Da diese App rein im Browser läuft, liegt das Secret sichtbar im Code — nicht öffentlich teilen/hosten.

Hinweis: Die QR-Codes lassen sich mit jeder Kamera-App oder der Spotify-App scannen und öffnen den Song direkt. Eine Erkennung durch die offizielle Hitster-App ist technisch nicht möglich, da diese nur ihre eigene, fest hinterlegte Songdatenbank kennt — dieser Generator erstellt daher eigenständige Karten im selben Spielprinzip. Der Zufallsgenerator wählt Songs aus einer öffentlichen YouTube-Playlist (über die YouTube Data API v3) und holt Jahr + exakten Track-Link über die Spotify-API. Dafür ist kurz Internet nötig, plus ein eigener, kostenloser YouTube-API-Key und Spotify Client-ID/Secret.
